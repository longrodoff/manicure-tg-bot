# 💅 Telegram-бот для студии маникюра (CRM & Booking)

Современный Telegram-бот для малого бизнеса, разработанный для автоматизации записи клиентов, выдачи прайс-листа и сбора базы контактов.

## 🌟 Функционал
- **Главное меню**: Удобная навигация через Reply-кнопки (Прайс, Запись, Контакты).
- **Инлайн-запись**: Удобные прозрачные кнопки для связи с администратором.
- **Мини-CRM**: Автоматическое логирование действий пользователей и сохранение базы клиентов в файл `clients.csv`.
- **Защита от спама**: Обработка некорректного текстового ввода и возврат пользователя в воронку продаж.

## 🛠 Стек технологий
- **Язык**: Python
- **Фреймворк**: aiogram 3.x (асинхронная разработка)
- **База данных**: локальная CSV
- **Безопасность**: `python-dotenv` для защиты секретных API-ключей.

## 🚀 Как запустить локально

1. Склонируйте репозиторий:
```bash
git clone https://github.com/longrodoff/manicure-tg-bot.git
```

2. Установите необходимые библиотеки:
```bash
pip install aiogram python-dotenv
```

3. Создайте файл `.env` в корневой папке проекта и добавьте туда токен вашего бота:
```env
BOT_TOKEN=ваш_токен_от_botfather
```

4. Запустите скрипт:
```bash
python main.py
```
